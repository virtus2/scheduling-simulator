# scheduling-simulator
Operating System Scheduling Simulator.  
2021년 1학기 운영체제 강의 과제 스케줄링 시뮬레이터입니다.

### 구성
- Makefile  
컴파일 하기 위한 파일

- sched.c  
스케줄링 알고리즘 관련 소스파일

- sched_test.c  
sched.c에서 구현한 알고리즘을 수행할 테스트 코드

- implement-and-result.pdf  
구현 방법 및 테스트 결과 관련 문서
